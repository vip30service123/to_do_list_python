# to_do_list_python


### Description
A user want to make a to-do list for himself

##### Details
User open the web, if cached, go to page, if not, log-in </br>
The website show a list of to-do tasks </br>
User have 3 options: </br>
- Create a task </br>
- Update the task </br>
- Delete the task </br>
User log-out </br>
